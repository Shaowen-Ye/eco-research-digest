# eco-research-digest

生态研究前沿速览 | Ecology & Environmental Research Digest

## 简介

本仓库收录生态与环境领域最新研究论文速递，按主题分类整理，提供标题、摘要概括与解读点评，方便快速浏览和筛选感兴趣的研究方向。

## 检索范围

- **期刊**：覆盖生态学、环境科学、海洋与淡水、保护生物学、气候变化、农业与植物等多个学科方向的主流期刊及预印本平台
- **筛选规则**：综合顶刊按生态环境关键词过滤，专业期刊全量收录
- **更新频率**：每周发布

## 每期内容

每期按主题归组，每篇包含：

1. 英文标题 + 中文翻译
2. 期刊名称、发表日期
3. 摘要图（Graphical Abstract，如有）
4. 摘要概括
5. 解读点评
6. APA 题录（含 DOI 链接）
